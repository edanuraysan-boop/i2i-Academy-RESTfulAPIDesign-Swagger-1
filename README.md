# RESTful API Design & Swagger

## Proje Hakkında

Bu proje, i2i Academy RESTful API Design & Swagger ödevi kapsamında geliştirilmiştir.

Spring Boot kullanılarak Oracle XE veritabanına bağlı katmanlı bir REST API geliştirilmiştir. Uygulamada CRUD (Create, Read, Update, Delete) işlemleri gerçekleştirilmiş ve Swagger UI ile API testleri yapılmıştır.

## Kullanılan Teknolojiler

- Java 17
- Spring Boot
- Spring Data JPA
- Oracle XE
- Swagger (OpenAPI)
- Maven
- Lombok

## Proje Yapısı

- Controller
- Service
- Repository
- DTO (Request & Response)
- Entity

## Gerçekleştirilen İşlemler

- Yeni müşteri ekleme
- ID ile müşteri görüntüleme
- Tüm müşterileri listeleme
- Müşteri güncelleme
- Müşteri silme

## Veritabanı

Veritabanı olarak Docker üzerinde çalışan Oracle XE kullanılmıştır.
